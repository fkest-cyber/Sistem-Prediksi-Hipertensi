1️⃣

Load Model

Buka file model_hipertensi.pkl menggunakan pickle

with open('model_hipertensi.pkl', 'rb') as file: model = pickle.load(file)
2️⃣

Data Pasien

Siapkan data pasien dalam bentuk dictionary dengan 10 fitur

pasien = { 'Age': 52, 'Salt_Intake': 11.5, 'Stress_Score': 7, ... }
3️⃣

DataFrame

Konversi dictionary menjadi pandas DataFrame

df = pd.DataFrame([pasien])
4️⃣

Prediksi

Gunakan model untuk prediksi risiko hipertensi

hasil = model.predict(df)[0] proba = model.predict_proba(df)[0]
5️⃣

Hasil

Interpretasi dan tampilkan hasil prediksi dengan rekomendasi

if hasil == 1: print("⚠️ BERISIKO") else: print("✓ TIDAK BERISIKO")
